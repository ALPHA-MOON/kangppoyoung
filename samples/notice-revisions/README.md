# 개정본 등록 · diff · RAG 검색 반영 테스트 샘플

공고/참고자료의 **개정본 등록 → diff 확인 → 검색(RAG) 반영**을 직접 검증하기 위한 샘플 PDF와 질문/답변 세트.

> 생성 스크립트: [`make_samples.py`](make_samples.py) (`python make_samples.py`로 재생성, Malgun Gothic 임베드).

## 파일

| 파일 | 용도 |
|------|------|
| `regulation_v1.pdf` / `regulation_v2.pdf` | 공고(regulation) 원본/개정본 |
| `reference_v1.pdf` / `reference_v2.pdf` | 참고자료(reference) 원본/개정본 |

각 PDF는 **3페이지**다. 1·3페이지는 v1/v2가 **동일**(diff에서 '동일' 블록), **2페이지만 개정**되어 diff가 깔끔하게 보인다.

## 무엇이 바뀌나 (2페이지 개정 내용)

**공고(regulation) — v1 → v2**
- 지원한도: `10억원` → **`30억원`** (변경)
- **`제5조의2(수출기업 우대한도)` 신설**: 수출기업은 최대 **50억원**까지 우대 (추가, v1엔 없음)

**참고자료(reference) — v1 → v2**
- 제출 서류에 **`수출실적증명서` 추가**
- **`[참고2의2] 온라인 제출 안내` 신설**: **온라인 접수만 허용** (추가, v1엔 없음)

## 질문 / 답변 세트 (개정판 반영 확인용)

검색 시 **개정판(v2)** 내용으로 답/근거가 나와야 통과.

| # | 질문 | v1 기준(원본) | **v2 기준(개정판, 기대)** |
|---|------|---------------|--------------------------|
| Q1 | 정책자금 지원한도는 얼마인가요? | 10억원 | **30억원** |
| Q2 | 수출기업 우대 한도는 얼마인가요? | (조항 없음) | **최대 50억원** (제5조의2) |
| Q3 | 신청 서류에 수출실적증명서가 필요한가요? | 미포함 | **포함됨** |
| Q4 | 신청 서류는 온라인으로만 접수하나요? | (명시 없음) | **온라인 접수만 허용** ([참고2의2]) |

- Q1·Q2 → 공고(regulation), Q3·Q4 → 참고자료(reference).
- Q2·Q4는 v1에 **없던 조항**이라, 검색이 이 내용을 답하면 "개정판이 RAG에 반영됨"이 명확히 증명된다.

## 직접 해보기 (UI)

1. 좌측 **정책 자금 공고 → 공고** → 우상단 **`+ 개정본 등록`**.
2. `regulation_v1.pdf` 업로드 → 전처리 완료 → 검토 화면에서 **시행일** 입력(예: `2026-02-01`) → **승인 후 등록**.
   - 시행일은 **현재 최신본 이후**여야 한다(과거 시행일 등록 금지). 부트스트랩 v1(2026-01-01)보다 이후로.
3. 다시 `+ 개정본 등록` → `regulation_v2.pdf` → 시행일 `2026-03-01` → 승인.
4. 우상단 **버전 드롭다운**에서 최신본 선택 → 본문에서 **2페이지** diff 확인:
   `10억원`(빨강 삭제) → `30억원`·`수출기업 우대한도 50억원`(초록 추가).
5. 좌측 **통합 검색**에서 **Q1/Q2** 질문 → 답변·근거에 **30억 / 50억(수출기업 우대)**가 나오면 개정판이 검색에 반영된 것.
   - 검색 반영은 등록 후 **비동기 재색인**이 끝난 뒤다(보통 수십 초). openai 모드면 자연어 답변, 오프라인 모드면 근거(조항) 나열.
6. **참고자료**도 동일하게 `reference_v1.pdf` → `reference_v2.pdf` 등록 후 **Q3/Q4**로 확인.

> 참고: 개정본 등록 경로(`/revisions`)는 nginx에서 **사내/사설망 전용**(공개 IP 차단, PRD §9)이다.
> Docker 배포에서는 클라이언트가 브리지 게이트웨이(172.16/12)로 보이므로 `nginx/nginx.conf`가 사내(10/8)·
> 사설(172.16/12, 192.168/16)·로컬 대역을 허용한다. 설정 변경 후에는 `docker compose restart nginx`로 반영한다.
